# 📚 Study Planner

A simple and efficient **Study Planner** web application that helps students organize their study schedule, manage tasks, and stay productive.

## 🚀 Live Demo

🔗 Live Website: YOUR_LIVE_LINK

## 📌 Features

* 📅 Plan and organize study sessions
* ✅ Track completed tasks
* 📝 Add and manage study topics
* 🎯 Improve productivity and time management
* 💻 Simple and clean user interface

## 🛠️ Built With

* **HTML5** – Structure of the application
* **CSS3** – Styling and layout
* **JavaScript (Vanilla JS)** – Functionality and interaction

## 📂 Project Structure

```
study-planner
│
├── index.html   # Main HTML file
├── style.css    # Styling for the app
└── app.js       # JavaScript functionality
```

## 📦 Installation

1. Clone the repository

```
git clone https://github.com/your-username/study-planner.git
```

2. Open the project folder

3. Run `index.html` in your browser
